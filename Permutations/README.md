
# PERMUTATIONS

## Problem Statement :
  Load a csv file and print the following pattern. 
  aix, aiy, ajx, ajy, bix, biy, bjx, bjy, cix, ciy, cjx, cjy Expected CLI Input
  ./executable input.csv
  
## Requirement : 
> g++17

## How to use?
 - Compile :
 
 ``` $ g++ permutations.cpp -o executable```
 - Run :
  
 ``` $ ./executable input.csv ```
